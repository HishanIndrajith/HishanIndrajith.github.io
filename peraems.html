Project Decription

Pera EMS is a Computerized System for University Health Center of University of Peradeniya. As it has no such electronic system up to now and uses manual data storing system, We are implementing this electronic System. Actually it is an academic group project for CO226 – Database System course, 4th semester, Bsc. Computer Engineering course. We have not implemented it yet in the Health Center.
